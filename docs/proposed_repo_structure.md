# Proposed Repository Structure

This is the target structure proposal based on a full scan of the current repository.

Notes:
- Excludes generated/vendor/runtime files such as `node_modules`, `dist`, `__pycache__`, `logs`, and coverage artifacts.
- Keeps backend, frontend, docs, and infra separated at top level.
